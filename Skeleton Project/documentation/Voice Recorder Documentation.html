Voice Recorder

The Voice Recorder sample project implements a voice recorder that performs audio sampling and playback by using the audio input and output channels on the NI myRIO. This sample project also streams the sampled audio to a desktop computer to display on waveform graphs. You can record the sampled audio for later playback. In order to achieve optimal performance, this sample project uses a custom FPGA VI to perform audio signal processing on the NI myRIO FPGA target.

This sample project is based on the myRIO Custom FPGA Project template. Refer to the myRIO Custom FPGA Project template and its documentation, available from the Create Project dialog box, for information about how the template works.

System Requirements

Development System

  • LabVIEW Full or Professional Development System
  • LabVIEW for myRIO Module
  • LabVIEW Real-Time Module
  • LabVIEW FPGA Module

Hardware

  • NI myRIO
  • A microphone
  • Headphones or a speaker

Overview

This sample project consists of the following VIs:

  • RT Main.vi—Sets the filtering frequency range, acquires filtered audio data from the FPGA target, and sends the filtered data to the desktop using network streams. This VI runs on the NI myRIO.
  • Desktop Main.vi—Receives the filtered audio data from network streams and displays the data on waveform graphs. This VI runs on the desktop computer.

Running This Sample Project

Before you run this sample project, connect the microphone to the Audio In connector on the NI myRIO-1900 or the A/AI0 channel on the NI myRIO-1950. Connect the headphones or speaker to the Audio Out connector on the NI myRIO-1900 or the A/AO0 channel on the NI myRIO-1950.

Complete the following steps to run this sample project:

  1. In the Project Explorer window, open myRIO-1900/1950 Customized FPGA.vi under the FPGA target.
  2. Press the <Ctrl> key and click Run to compile a custom FPGA bitfile.
  3. Open RT Main.vi.
  4. Click Run.
  5. Speak into the microphone and listen to the output from the headphones or speaker.
  6. Select different values in Frequency Selector to generate different output audio signals.
  7. Open Desktop Main.vi.
  8. Specify the IP address of the NI myRIO in Target IP Address.
  9. Click Run. The Connected? indicator returns TRUE when Desktop Main.vi receives data from the network streams.
  10. Speak into the microphone and observe the waveform graphs of the output audio signal.
  11. (Optional) Click Record to record the audio signal as a .wav file.
  12. Click Stop on both VIs to stop the application.

Running the RT Main VI Only

Complete the following steps to run RT Main.vi without running Desktop Main.vi:

  1. In the Project Explorer window, open myRIO-1900/1950 Customized FPGA.vi under the FPGA target.
  2. Press the <Ctrl> key and click Run to compile a custom FPGA bitfile.
  3. Open RT Main.vi.
  4. Click Run.
  5. Speak into the microphone and listen to the output from the headphones or speaker.
  6. Select different values in Frequency Selector to generate different output audio signals.
  7. Click Stop to stop the application.
